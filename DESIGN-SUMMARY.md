# Design summary

- Design concept: Assistant directory
- Layout personality: clean Swiss index
- Typography: Helvetica Neue headings with Arial body text
- Color palette: primary #0f172a, secondary #cbd5e1, accent #2563eb, background #ffffff, text #020617, muted #64748b
- Shape language: Minimal use of containers
- Visual motif: index numbers and strict alignment
- Unique layout choices: first sections use Service index, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
